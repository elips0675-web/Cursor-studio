# Логика работы функции «Автопоиск»

Этот документ описывает, как работает алгоритм "Автопоиска" в приложении SwiftMatch для подбора наиболее релевантных анкет.

## Основной принцип

Алгоритм работает в два этапа:
1.  **Жесткая фильтрация:** Сначала отсеиваются все анкеты, которые не соответствуют базовым критериям поиска.
2.  **Умное ранжирование (скоринг):** Оставшиеся анкеты получают баллы (очки) на основе совпадения по более "мягким" критериям, таким как цель знакомства и интересы. Анкеты с наибольшим количеством баллов показываются первыми.

---

## Этап 1: Жесткая фильтрация

Из общего списка пользователей исключаются все, кто не соответствует следующим обязательным параметрам, заданным в фильтре:

*   **Пол:** Если вы ищете "Мужчин" или "Женщин", будут отобраны только анкеты соответствующего пола.
*   **Возраст:** Пользователи, чей возраст не попадает в указанный вами диапазон (например, 25-35 лет), будут исключены.
*   **Город:** Если выбран конкретный город, пользователи из других городов будут отфильтрованы.
*   **Расстояние:** Пользователи, находящиеся дальше указанного расстояния (в км), также исключаются.

На этом этапе мы получаем список потенциальных кандидатов.

---

## Этап 2: Умное ранжирование (скоринг)

Каждый кандидат из отфильтрованного списка получает "очки совместимости". Чем больше очков, тем выше анкета будет в результатах поиска.

Баллы начисляются по следующим правилам:

1.  **+1000 очков за совпадение цели знакомства:**
    *   Это самый важный критерий. Если цель знакомства пользователя совпадает с вашей (например, оба ищут "Серьезные отношения"), его анкета получает огромный приоритет.

2.  **+100 очков за каждый общий интерес:**
    *   За каждый интерес, который совпадает с вашими, анкета получает дополнительные баллы. Чем больше общих интересов, тем выше балл.

3.  **Финальная проверка кандидата:**
    *   Чтобы попасть в итоговый список, анкета должна удовлетворять **хотя бы одному** из двух условий:
        *   Иметь совпадающую с вашей цель знакомства.
        *   ИЛИ иметь хотя бы один общий с вами интерес.

    Это сделано для того, чтобы не показывать вам совершенно нерелевантные анкеты, у которых нет с вами ничего общего.

## Итоговый результат

После подсчета очков все анкеты-кандидаты сортируются по убыванию их рейтинга. В результате на первых местах вы видите:
1.  Пользователей с совпадающей целью знакомства, отсортированных по количеству общих интересов.
2.  Затем — пользователей с другой целью знакомства, но с которыми у вас есть общие интересы, также отсортированных по их количеству.

Такой подход позволяет находить наиболее подходящих людей, но при этом не упускать из виду интересные профили, с которыми у вас есть точки соприкосновения.
